# YTP+ Deluxe Edition Tutorial (Older Windows)

## Supported Targets
- Windows XP, Longhorn, Vista (beta/RTM)
- Windows 7 (beta/RTM), Windows 8 (beta/RTM), Windows 8.1

## Setup Notes
- Install the highest .NET Framework supported by your OS.
- Use a compatible FFmpeg build for your Windows version.
- Keep assets on local disks to avoid path/permission issues.

## Workflow
1. Open the project in the supported Visual Studio version for your OS.
2. Add local sources using the Source browsers.
3. Keep effect toggles conservative for stability.
4. Render and export with FFmpeg.
