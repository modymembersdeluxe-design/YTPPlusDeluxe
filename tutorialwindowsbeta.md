# YTP+ Deluxe Edition Tutorial (Windows Beta Releases)

## Beta OS Guidance
- Use stable builds of FFmpeg compatible with the beta OS.
- Expect missing codecs/drivers; test with small clips first.
- Prefer local assets and short render jobs.

## Quick Steps
1. Install the latest supported .NET Framework for your beta build.
2. Configure FFmpeg/FFprobe paths in the app.
3. Add sources, preview clips, and render small batches.
