# YTP+ Deluxe Edition Tutorial (Old Windows Betas)

## Target Builds
- Windows Longhorn
- Windows 7 beta
- Windows 8 beta
- Windows 10 beta

## Practical Tips
- Use the most compatible .NET runtime available for each beta.
- Keep FFmpeg builds aligned to the OS architecture.
- Preview and render with minimal effects and short clips.

## Recommended Flow
1. Add a few local video sources.
2. Enable only one or two effects.
3. Render a short output to validate the pipeline.
